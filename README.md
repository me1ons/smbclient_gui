# smbclient_gui
impacket中smbclient的GUI实现。

## 使用方式
GUI版与原生版本一样，启动只需添加`-gui`,像下面这样。
```
python3 smbclient_gui.py ./administrator@127.0.0.1 -hashes :00affd88fa323b00d4560bf9fef0ec2f -gui
```
